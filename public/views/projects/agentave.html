Agent Ave

HTML5, CSS3, jQuery, Bootstrap


Agent Ave is the latest productivity solution for Independent Insurance Agents to save time & drive sales.

Agent Ave takes the hassle out of managing carrier data such as underwriting guidelines, ID/Passwords, industry contacts, sales reports and more. All data is accessible to all staff in one central location. Managing carrier data has never been easier.



Agent Ave brought me on board to develop an interactive and smooth front-end experience for their customers. The project required solutions to produce a website that matched the detailed designs and interactivity concepts provided by Agent Ave whilst remaining fast to use across a wide range of devices.



Agent Ave is a mobile responsive web page.



Agent Ave needed a Customer page to display their case-studies. I built this page as a 3 column row using Bootstrap.



Agent Ave wanted an Enterprise page to show their plans and pricing for their services. I built this page as a 3 column row using Bootstrap.



Conclusion

Agent Ave is a culmination of my front-end development skills, designing and develop a smooth landing page with responsive designs. Feel free to visit the website at www.agentave.com
